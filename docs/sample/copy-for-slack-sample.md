# 1. Headings
# Heading 1 (H1)
## Heading 2 (H2)
### Heading 3 (H3)
#### Heading 4 (H4)

# 2. Text Formatting
This is plain text.
*Italic* or _Italic_
**Bold** or __Bold__
~~Strikethrough~~
==Highlight==
`Inline Code`

> Blockquote
> Second line blockquote

# 3. Lists
* Unordered list 1
* Unordered list 2
	* Sub-list 2-1
	* Sub-list 2-2

1. Ordered list 1
2. Ordered list 2
3. Ordered list 3

# 4. Task List
- [x] Completed item
- [ ] In progress item
- [ ] Pending item

# 5. Links & Images
* **Internal link**: [[Note Name]]
* **Alias link**: [[Note Name|Display Text]]
* **External link**: [Obsidian Official Site](https://obsidian.md)
* **Image**: `![[image.png]]` (image in vault)
