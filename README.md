# Copy for Slack

Obsidian에서 선택한 텍스트를 **Slack에 맞는 포맷으로 변환**하여 클립보드에 복사하는 플러그인입니다.

Obsidian 마크다운을 그대로 Slack에 붙여넣으면 포맷이 깨지는 문제를 해결합니다.

## 사용법

### 바로 복사 (추천)
1. 텍스트 선택
2. `Alt+Shift+C`
3. Slack에 붙여넣기

### 미리보기 후 복사
1. 텍스트 선택
2. `Cmd+P` → "Copy as Slack format (미리보기)" 실행
3. 변환 결과 확인 후 "클립보드에 복사" 클릭

### 사이드바 아이콘
왼쪽 사이드바의 💬 아이콘을 클릭하면 미리보기 모달이 열립니다.

## 단축키

| 기능 | 기본 단축키 | 설명 |
|------|-----------|------|
| 바로 복사 | `Alt+Shift+C` | 미리보기 없이 즉시 클립보드에 복사 |
| 미리보기 | 없음 (설정에서 지정) | 원본/변환 비교 모달 표시 |

단축키는 **Obsidian 설정 → 단축키**에서 "Copy for Slack"을 검색하여 변경할 수 있습니다.

## 변환 규칙

### 텍스트 포맷팅

| Obsidian | Slack | 변환 |
|----------|-------|------|
| `**굵게**` | **굵게** | HTML `<b>` 태그로 변환 |
| `*기울임*` | *기울임* | HTML `<i>` 태그로 변환 |
| `~~취소선~~` | ~~취소선~~ | HTML `<s>` 태그로 변환 |
| `` `코드` `` | `코드` | HTML `<code>` 태그로 변환 |
| `==하이라이트==` | **하이라이트** | 볼드로 변환 |

### 구조 요소

| Obsidian | Slack | 변환 |
|----------|-------|------|
| `# 헤딩` ~ `###### 헤딩` | **헤딩** | 볼드로 변환 |
| `[링크](url)` | [링크](url) | HTML `<a>` 태그로 변환 |
| `![이미지](url)` | [이미지](url) | 링크로 변환 |
| `[[위키링크]]` | 위키링크 | 텍스트만 남김 |
| `[[노트\|별칭]]` | 별칭 | 별칭 텍스트만 남김 |
| `![[임베드]]` | 임베드 | 텍스트만 남김 |

### 리스트

| Obsidian | Slack | 변환 |
|----------|-------|------|
| `- 항목` / `* 항목` | • 항목 | HTML `<ul><li>` 구조 (중첩 지원) |
| `1. 항목` | 1. 항목 | HTML `<ol><li>` 구조 |
| `- [ ] 할 일` | ☐ 할 일 | 이모지로 변환 |
| `- [x] 완료` | ☑ 완료 | 이모지로 변환 |

### 기타

| Obsidian | Slack | 변환 |
|----------|-------|------|
| `> 인용문` | 인용문 | HTML `<blockquote>` 변환 |
| `> [!NOTE] 제목` | NOTE: 제목 | callout → 인용문 |
| `$수식$` | `수식` | 인라인 코드로 변환 |
| `$$수식$$` | 코드블록 | 코드블록으로 변환 |
| `%% 주석 %%` | (제거) | Obsidian 주석 삭제 |
| `---` | ─────── | 유니코드 구분선 |
| 테이블 | 텍스트 | `헤더: 값` 형태로 변환 |
| 각주 `[^1]` | (인라인) | 본문에 병합 |
| 백슬래시 이스케이프 `\.` | `.` | 이스케이프 제거 |

## 설치

### Git Clone (추천)

```bash
cd <vault경로>/.obsidian/plugins/
git clone https://gitlab.kolonfnc.com/silentc1/obsidian/copy-for-slack.git
```

Obsidian 재시작 → 설정 → 커뮤니티 플러그인 → "Copy for Slack" 활성화

### 설치 스크립트

```bash
bash install.sh ~/private_vault
```

### 업데이트

```bash
cd <vault경로>/.obsidian/plugins/copy-for-slack
git pull
```

Obsidian 재시작으로 적용.

## 개발

```bash
# 의존성 설치
npm install

# 개발 모드 (파일 변경 감지)
npm run dev

# 프로덕션 빌드
npm run build

# 테스트
npm test

# vault에 배포
npm run deploy
```

## 기술 구현

- **클립보드**: HTML 리치 텍스트(`text/html`)로 저장하여 Slack 붙여넣기 시 포맷팅 유지
- **변환 방식**: 정규식 기반 순차 변환 (코드블록 보호 → 전처리 → 인라인 변환 → 블록 변환 → 복원)
- **리스트**: 들여쓰기 레벨 파싱 → 중첩 `<ul>/<ol><li>` HTML 구조 생성
- **빌드**: esbuild 번들링, TypeScript
